Currently, Kaikki supports 15 wiktionary editions (en, zh, nl, fr, el, de, it, ja, ko, ku, ru, pl, pt, es, and th), so only dictionaries including these languages are available.

If the language you want isn't here, or you would like to see an improvement to a dictionary, please open an issue.

Some of the dictionaries listed here are small; rather than decide on a lower bound for usefulness they are all included here.

Languages are referred to by their shortest ISO code (ISO 639-1 where available, ISO 639-3 where not)

IPA Dictionaries

These dictionaries contain the International Phonetic Alphabet (IPA) transcriptions for the headwords. There are two types of IPA dictionaries:

  1. IPA dictionaries from a single wiktionary edition - e.g. en-de contains IPA transcriptions for English headwords from the German wiktionary edition,
  2. Merged IPA dictionaries from all 15 supported editions. These have more terms covered but not all the entries might be formatted the same way.
